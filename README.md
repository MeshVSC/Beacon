# Beacon
A curated directory of tools, inspiration, libraries, templates, fonts, and assets for builders and creatives.

## Features
- **161 curated tools** across multiple categories.
- **Search functionality** to find exactly what you need.
- **Dark/Light mode** support.
- **Minimalist UI** designed for speed and clarity.

## Project Structure
- `index.html`: The landing page with high-level categories and hero search.
- `beacon.html`: The main directory page with a searchable table of all tools.
